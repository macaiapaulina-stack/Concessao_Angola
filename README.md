# WebGIS Concessão

Mapa web em Leaflet com camadas de Concessões e Poços.

Publicação no GitHub Pages:
- URL (após o primeiro deploy): `https://macaiapaulina-stack.github.io/Concessao_Angola/`

Como publicar rapidamente (Windows):
1. Abra a pasta do projeto
2. Dê duplo-clique em `publish-github.bat`
3. Aguarde o envio (`git push`)
4. O GitHub Actions publicará automaticamente o site (workflow em `.github/workflows/deploy-pages.yml`)

Caso prefira ativar Pages por branch manualmente:
- Settings → Pages → Source: Deploy from a branch → `main` / `(root)`
